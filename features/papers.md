# การแลกกระดาษ

✨ **ข้อดีของการแลกกระดาษ** ✨ 

:+1: ลดขั้นตอนการจัดส่งกรมธรรม์ - สามารถพิมพ์กรมธรรม์ให้ลูกค้าได้ทันที ลดระยะเวลารอเอกสาร
    
:+1: เพิ่มความรวดเร็วในการทำธุรกรรม - ระบบหักเงินอัตโนมัติจากค่ากระดาษ ช่วยให้การซื้อ พ.ร.บ. เป็นไปอย่างรวดเร็ว
    
:+1: ตอบโจทย์ความต้องการของลูกค้า - สร้างความมั่นใจให้ผู้เอาประกัน โดยเฉพาะผู้ที่ต้องการถือกรมธรรม์ตัวจริงไว้ใช้งาน

## ขั้นตอนแลกกระดาษ

หลังจากลงชื่อเข้าสู่ระบบ ผู้ใช้งานจะเข้าสู่หน้าหลักที่มีเมนูให้เลือกใช้งานหลากหลาย การแลกกระดาษ สามารถกดจากแถบเมนูด้านข้าง หรือคลิกปุ่มลัดแลกกระดาษ ตามที่แสดงในรูปภาพ.

### วิธีการรับกระดาษ

- กดเลือกช่องทาง "รับทางไปรษณีย์" ระบบจะแสดงส่วน **การจัดส่งกระดาษ** ให้ทำรายการ

### การจัดส่งกระดาษ

- ช่องทางการจัดส่ง: ระบบจะเลือกใช้บริการจัดส่งผ่าน DHL โดยอัตโนมัติ
- ค่าจัดส่ง: อัตราค่าจัดส่งปัจจุบันอยู่ที่ 30 บาท (ค่าจัดส่งอาจมีการเปลี่ยนแปลงตามนโยบายหรือโปรโมชั่นในอนาคต)

{% hint style="info" %}
**หมายเหตุ:** ฟรี ค่าจัดส่งเมื่อแลกกระดาษเกิน 2,000 บาทขึ้นไป
{% endhint %}

### ที่อยู่สำหรับจัดส่ง

- กรณีไม่ต้องการแก้ไขที่อยู่ หรือ เปลี่ยนที่อยู่จัดส่งใหม่ กดปุ่ม "บันทึกข้อมูลการรับกระดาษ" เพื่อดำเนินการในขั้นตอนถัดไป
