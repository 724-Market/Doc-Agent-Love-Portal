# การแลกกระดาษ

**ข้อดีของการแลกกระดาษ**

- ลดขั้นตอนการจัดส่งกรมธรรม์
  - สามารถพิมพ์กรมธรรม์ให้ลูกค้าได้ทันที ลดระยะเวลารอเอกสาร
- เพิ่มความรวดเร็วในการทำธุรกรรม
  - ระบบหักเงินอัตโนมัติจากค่ากระดาษ ช่วยให้การซื้อ พ.ร.บ. เป็นไปอย่างรวดเร็ว
- ตอบโจทย์ความต้องการของลูกค้า
  - สร้างความมั่นใจให้ผู้เอาประกัน โดยเฉพาะผู้ที่ต้องการถือกรมธรรม์ตัวจริงไว้ใช้งาน
