# เริ่มต้นแจ้งงาน พ.ร.บ.

หลังจากลงชื่อเข้าสู่ระบบ ผู้ใช้งานจะเข้าสู่หน้าหลักที่มีเมนูให้เลือกใช้งานหลากหลาย การแจ้งงาน พ.ร.บ. สามารถกดจากแถบเมนูด้านข้าง
